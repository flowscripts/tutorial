What is HTML?

HTML stands for Hypertext Markup Language, and is used to create about 96% of all sites. Even FlowScripts (this website) is coded in HTML!
Everything is made up of elements, they include a start tag, content, and an end tag.

An Element:

<p>Hello World</p>

The text saying "Hello World" will be visible on your website.

However, the tags <p>, and</p>, will be hidden.

! Don't forget the forward slash in the end tag !

CSS and JavaScript

CSS (Cascading Style Sheets), is used to style your website; it can change the color of text, or even animate objects.
JavaScript is used to make your site work, example: if you click on a button, it will show a pop-up.
